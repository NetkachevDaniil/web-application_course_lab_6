# ЛР6 Flask

В работе доработан «Образовательный портал»: добавлена модель отзывов `Review` и миграция таблицы `reviews`, реализованы вывод 5 последних отзывов на странице курса, отдельная страница всех отзывов с пагинацией и сортировкой (новые/положительные/отрицательные), форма создания отзыва (оценка + текст) на обеих страницах, блокировка повторного отзыва от одного пользователя и пересчет рейтинга курса через обновление `rating_sum` и `rating_num`.
