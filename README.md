# ЛР6: Отзывы к курсам (Образовательный портал)

## Что реализовано

1. Добавлена модель `Review` в `app/models.py`:
   - `id` — PK
   - `rating` — целое число 0..5
   - `text` — текст отзыва
   - `created_at` — дата создания
   - `course_id` — FK на курс
   - `user_id` — FK на пользователя
   - связи `review.course`, `review.user`
   - уникальность пары `(course_id, user_id)`, чтобы один пользователь оставлял только один отзыв

2. Добавлены связи в существующие модели:
   - `Course.reviews`
   - `User.reviews`

3. Страница курса `courses.show` доработана:
   - отображаются 5 последних отзывов;
   - добавлена кнопка «Все отзывы»;
   - внизу добавлена форма отзыва (или уже оставленный отзыв пользователя).

4. Добавлена страница всех отзывов:
   - маршрут: `/courses/<int:course_id>/reviews`
   - пагинация
   - фильтр сортировки:
     - сначала новые (по умолчанию),
     - сначала положительные,
     - сначала отрицательные
   - при переходах по страницам параметр сортировки сохраняется.

5. Добавлено создание отзыва:
   - маршрут: `POST /courses/<int:course_id>/reviews/create`
   - доступ только авторизованным
   - форма содержит:
     - селектор оценки (значения 5..0, подписи как в ТЗ)
     - текст отзыва
   - если пользователь уже оставил отзыв, новая форма не показывается.

6. Пересчет рейтинга курса:
   - после создания отзыва пересчитываются `rating_sum` и `rating_num` на основе фактических отзывов.

## Что проверить вручную

1. Создать/открыть курс.
2. Оставить отзыв авторизованным пользователем.
3. Убедиться, что:
   - отзыв появился на странице курса;
   - отзыв появился на странице «Все отзывы»;
   - рейтинг курса обновился;
   - повторно форма этому пользователю не показывается.
4. Переключить сортировку на странице всех отзывов и пройти по страницам пагинации.
